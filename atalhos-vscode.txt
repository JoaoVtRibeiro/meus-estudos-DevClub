Reiniciar o vscode : CTRL + SHIFT + P -> reiniciar/reload

Montar uma estrutura inicial : html:5

Cria um texto aleatório : lorem

Ajustar o código : shift + alt + f

Fazer um comentário : shift + alt + a

Olhar a configurações (No Navegador) : Botão direito -> Inspect/ Inspecionar Elemento

Mostrar arquivos ocultos (Windows) = ctrl + shift + .

Selecionar uma mesma palavra que se repete varias vezes no código (Uma por vez) = ctrl + d 

Selecionar uma mesma palavra que se repete varias vezes no código (Todas de uma vez) = ctrl + shift + l
    - Após selecionar a palavra, segurar o shift e apertar "end" para selecionar toda a linha

Receber sugestões para completar o contexto (ex: informações do author em um package.json) = ctrl + espaço

Emojis: Windows + .

Copiar uma linha: shif + alt + (seta para cima ou baixo)
