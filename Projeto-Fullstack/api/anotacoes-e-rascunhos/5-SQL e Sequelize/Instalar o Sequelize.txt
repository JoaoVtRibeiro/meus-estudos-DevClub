Instalando o Sequelize:
    1. yarn add sequelize

    2. yarn add sequelize-cli -D
        - Uma biblioteca para auxiliar na criação e gerenciamento de alguns arquivos    

    3.  Informar qual banco de dados será utilizado
        -Link: https://sequelize.org/docs/v6/getting-started/
        - Comando: yarn add pg pg-hstore (postgres)
